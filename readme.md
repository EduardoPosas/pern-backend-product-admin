# PERN - TypeScript - Administrador de productos

REST API para administrar productos.

## Tecnologías

* Node
* Express
* TypeScript
* Sequelize


## Dependencias

### Desarrollo

* nodemon
* ts-node
* typescript (tsc para compilar ts a js)

### Producción

* express
* express and node types

## Comandos útiles

```cmd
tsc -w
nodemon dist/index.js
```
